## Eclipse 笔记：

在class里边输入`main`然后按下`CTRL + Space`然后回车`Enter`，就能快速生成

``` java
public static void main(String[] args){}
```

在main或者任意函数里边输入`sysout`或`sout`然后按下`CTRL + Space`，就能快速生成

``` Java
System.out.println();
```

快速上下复制一行：`CTRL + ALT + 上下方向键`

快速移动整行：`ALT + 上下方向键`
